Lab: Creating a Watson Studio Project with Jupyter Notebooks

Objective(s):

After completing this lab, you will be able to:

  • Use Watson Studio service
  • Create project in Watson Studio
  • Add an interactive python notebook to a project in Watson Studio

Pre-requisite

You need an IBM Cloud account to create a project in Watson Studio. If you don't have an account created already, click and open this link and follow the instructions, to create an IBM Cloud account.

Exercise - Create a project on Watson Studio

If you have not created a Watson service before proceed with Task 1, otherwise go to Task 2

Task 1: Create Watson Studio Service:

  1. Click here to go to the IBM Cloud Watson Studio page. You will see the screen in the figure below. Click on the Create button.


  1. Now click Launch IBM Cloud Pak for Data.


  1. Then click Go to IBM Cloud Pak for Data.


  1. Then click Continue.


Task 2: Open Watson Studio

  1. Go to the IBM Cloud Dashboard and click Services.
  1. When you click on Services, all your existing services will be shown in the list. Click the Watson Studio service you created:
  1. Then click Get Started.


Task 3: Create a Project

  1. Click on Create a project.


  1. On the Create a project page, click Create an empty project.


  1. Provide a Project Name and Description.


  1. You must also create storage for the project. Click Add


  1. On the Cloud Object Storage page, click Create.


  1. On the New project page, note that the storage has been added, click Refresh and then click Create.


Task 4: Adding a Notebook to the Project:

  1. Click on Assets > New asset.


  1. In the list of asset types, click Jupyter Notebook Editor.


  1. On the New notebook page, click Blank and then add a name and optional description for the notebook. Specify the language as Python and runtime environment. Click Create.

Just wait until the notebook appears. In case you are interested. The jupyter enterprise gateway has requested resources on the Kubernetes cluster IBM hosts for serving the jupyter kernel backing your notebook.

Now you're ready to code!

This concludes this tutorial.
